# axiom-pulse-replica

Pixel-perfect frontend replica of Axiom Trade Pulse — built with **Next.js 14**, **TypeScript (strict)**, **Tailwind CSS**, **Redux Toolkit**, **React Query**, and **Radix/shadcn/ui**.
